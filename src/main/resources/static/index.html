How It Works

3단계로 끝나는 면접 준비

복잡한 설정 없이, GitHub URL 하나만으로 시작하세요.

1

URL 입력

분석할 GitHub 저장소의 URL을 입력하세요. public 저장소라면 바로 시작됩니다.

2

AI 분석 & 질문 생성

Gemini AI가 코드를 읽고 품질·보안 이슈를 분석한 뒤, 내 프로젝트 맞춤 면접 질문을 자동으로 생성합니다.

3

답변 작성 & AI 리뷰

주니어·시니어·면접관 역할 중 선택하면 맞춤 질문 5개와 모범 답변을 제공합니다. 내 답변을 AI가 점수와 피드백으로 리뷰해 줍니다.
